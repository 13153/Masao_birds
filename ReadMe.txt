Masao bird(c)2017-2018

<概要> 
・MASAOはいつも落ち込んでいるので勝手に落ちていく 
・MASAOはテンションを上げて羽ばたくことができる 
・留年の壁は左方向へスクロールする 

このままではMASAOは留年してしまうので壁をうまく回避し、卒業させよう

<操作方法>
・Spaceを押してゲーム開始。
・タイミングよくSpaceを押してMASAOのテンションをあげよう
